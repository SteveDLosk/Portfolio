I also completed a machine learning challenge in natural language processing, making a language detector. By collecting language samples from wikipedia articles, and writing algorithms to count words, I was able to find the most frequently used words in the following languages: English, French, Spanish, and German. Next, by filtering duplicates (particularly between Spanish and French), I was able to create unique lists for each of these languages. Finally, given an input text, this algorithm counts each time one of the words from each of these language lists shows up. The largest sum is the language in question.
